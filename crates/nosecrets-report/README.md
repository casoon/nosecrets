# nosecrets-report

[![Crates.io](https://img.shields.io/crates/v/nosecrets-report.svg)](https://crates.io/crates/nosecrets-report)

Reporting and output formatting for [nosecrets](https://crates.io/crates/nosecrets-cli).

This crate provides text and JSON output for scan findings. For most use cases, install the CLI instead:

```bash
cargo install nosecrets-cli
```
