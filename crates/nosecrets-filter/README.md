# nosecrets-filter

[![Crates.io](https://img.shields.io/crates/v/nosecrets-filter.svg)](https://crates.io/crates/nosecrets-filter)

Filtering and allowlist logic for [nosecrets](https://crates.io/crates/nosecrets-cli).

This crate handles ignore patterns, path filtering, and inline suppression. For most use cases, install the CLI instead:

```bash
cargo install nosecrets-cli
```
