# nosecrets-core

[![Crates.io](https://img.shields.io/crates/v/nosecrets-core.svg)](https://crates.io/crates/nosecrets-core)

Core scanning engine for [nosecrets](https://crates.io/crates/nosecrets-cli).

This crate provides the file scanning and secret detection logic. For most use cases, install the CLI instead:

```bash
cargo install nosecrets-cli
```
