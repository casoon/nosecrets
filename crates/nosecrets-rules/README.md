# nosecrets-rules

[![Crates.io](https://img.shields.io/crates/v/nosecrets-rules.svg)](https://crates.io/crates/nosecrets-rules)

Rule definitions and parsing for [nosecrets](https://crates.io/crates/nosecrets-cli).

This crate provides the TOML rule format and built-in detection rules. For most use cases, install the CLI instead:

```bash
cargo install nosecrets-cli
```
